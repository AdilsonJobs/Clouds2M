How to Create a Auto-Scaling Groups

Made by Clouds2 with Scribe

This guide provides step-by-step instructions on how to create an auto-scaling group. By following these steps, you will be able to set up a system that automatically scales up and down based on demand, ensuring optimal performance and cost-efficiency.



1. Navigate to https://lad-z-mnode.clouds2africa.com/

2. Click "Menu"

3. Click "Auto-Scaling Groups"

4. Click "Create".

5. Fill in all fields

Tip: Define number of instance (MIN, Max or Desired) and others

6. Click this field.

7. Select the Subnet

8. Select "Launch Configuration"

9. Click "Next"

10. Select you Scalling Policy: CPU Utilization or Network

11. Define your target to start scalling "ex: 80"

12. In heath Check select "VM Health" or "Target Group"

13. Select the Target Group

14. Click the Termination Policies " Ex: oldest launch configuration"

15. Click "Finish"

16. Auto-sacalling was created...

17. Resume

18. Click "ASG-Demo"
