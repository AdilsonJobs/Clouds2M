How to Create a Project

Made by Clouds2 with Scribe

This guide provides a simple step-by-step process for creating a project on Clouds2. If you need help with project creation, this guide is a must-read.



1. Navigate to https://lad-z-mnode.clouds2africa.com/

2. Click "Menu"

3. Click "Account"

4. Click "Create Project"

5. Insert the "Project Name".

6. Click "Ok"

7. Project was created "Successfully"

8. Resume
