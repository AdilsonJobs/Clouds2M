How to Configure Security Groups

Made by Clouds2 with Scribe

This guide provides step-by-step instructions on how to configure security groups. By following these steps, users will be able to create and attach security groups to their instances, ensuring enhanced security and control over network traffic.



1. Navigate to https://for-z-mnode.clouds2brasil.com.br/

2. Click "Security Groups"

3. Click "Create"

4. Fill in all fields

5. Click "Add"

6. Click here.

7. Select IPv4 > Ingress and Egress > Any (To set a DMZ)

8. Click "Ok"

9. Click "Home"

10. Click "Instances"

11. Select a Instance

12. Click "Networks"

13. Select the network

14. Click in Security Group

15. Select the Security Group

16. Click "Ok"

17. Now you can see the Security Group attached
