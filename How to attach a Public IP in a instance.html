How to attach a Public IP in a instance

Made by Clouds2 with Scribe

This guide provides step-by-step instructions on how to attach a Public IP to an instance. Follow the instructions to successfully attach a Public IP.



1. Navigate to https://for-z-mnode.clouds2brasil.com.br/

2. Click "Instances"

3. Select the Instance

4. Click "Networks"

5. Click here.

6. Select "Elastic IP"

7. Click here.

8. Click "Ok"

9. Click "Ok"

10. Now you can see your Public IP attached
