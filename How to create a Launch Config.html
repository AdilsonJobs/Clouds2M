How to create a Launch Config

Made by Clouds2 with Scribe

This guide provides step-by-step instructions on how to create a Launch Config (Launch Configuration). Following this guide will enable you to easily set up and configure Launch Configurations for your instances, ensuring smooth and efficient deployment in the future.



1. Navigate to https://lad-z-mnode.clouds2africa.com/

2. Click here.

3. Click "Launch Config"

4. Click "+".

5. Type da launch config name ex:"LC-Demo01"

6. Select the instance type

7. Select the VPC

8. Select the Security Group

9. Select or create a key par or leave (optional)

10. Click "Next"

11. Click the "Snapshot" field.

12. Choose Snapshot

13. Click "Ok"

14. Launch Config was created "Action Succeeded"

15. Launch Config resume
