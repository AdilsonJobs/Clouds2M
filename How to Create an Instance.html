How to Create an Instance

Made by Clouds2 with Scribe

This guide provides step-by-step instructions on how to create an instance in a clouds2 platform. It covers everything from navigating to the platform's website to selecting the image, instance type, operating system, and other important settings. Following this guide will help users easily create their own instances for various purposes.



1. Navigate to https://for-z-mnode.clouds2brasil.com.br/

2. Click "Instances"

3. Click in Create

4. Select the Image

5. Select the "Instance Type":

6. Select Operating System from the list:

7. Click here.

8. Type the name for your Private Key

9. Click "Next"

10. Click "Save"

11. Click "Finish"

12. Click "Next"

13. Choose the size and type disk of your VM

14. Click "Next"

15. Select a Subnet

16. Select security group

17. Click "Next"

18. Click "Finish"
