How to Create a Target Group

Made by Clouds2 with Scribe

This guide provides step-by-step instructions on how to create a target group. By following these steps, you will be able to easily navigate to the necessary fields, enter the required information, and create a target group for your load balance.



1. Navigate to https://lad-z-mnode.clouds2africa.com/

2. Click "Menu"

3. Click "Target Groups"

4. Click "+".

5. Type Target Group name "TG-Demo" and other field

6. Click the "HTTP" or "TCP".

7. Click this button.

8. Enable sticky session if necessary and define time,

So that requests from a specific user are always directed to the same server in distributed environments.

9. Click "Next"

10. Click the "HTTP" or "TCP" field.

11. Click the "Traffic Port" field.

12. Click "Next"

13. Click "Add"

14. Click "Add"

15. Select the instances for you target and port

16. Click "Finish"

17. Target Group was Created

18. Target Group Resume
