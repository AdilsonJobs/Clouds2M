How to Create a VPC

Made by Clouds2 with Scribe

This guide provides step-by-step instructions on how to create a VPC with public and private subnets. It covers to configuring the subnets and security groups. If you need to set up a VPC with both public and private subnets, this guide is a valuable resource.



1. Navigate to https://lad-z-mnode.clouds2africa.com/

2. Click "Menu"

3. Click "VPCs"

4. Click Wizard

5. Click "VPC with Public and Private Subnets" or other

6. Wizard will create to you other VPC resources

7. Click "Next"

8. Fill in all fields

9. Click "Next"

10. Insert the "Private Subnet Name" and "CIDR"

11. Click "Next"

12. Insert the "Public Subnet Name" and "CIDR"

13. Click "Next"

14. Fill in all fields and add a elastic IP for NAT Gateway

15. Click in "+" to get new Elastic IP

16. Click "Finish"

17. VPC setup completed!

18. Click "Go To VPC"
