How to Create a Load Balancer

Made by Clouds2 with Scribe

This guide provides step-by-step instructions on how to create a load balancer. By following these steps, users will be able to easily set up a load balancer for their applications, improving reliability and scalability.



1. Navigate to https://lad-z-mnode.clouds2africa.com/

2. Click "Menu"

3. Click "Load Balancers"

4. Click "+".

5. Choose the LB Type "ALB or NLB"

6. Select the subnet

7. Select Auto Select to get Public IP for load balance

8. Click the "Standard or Managed" Security Groups .

9. Select the Security Group

10. Select the instance type for Load Balance

11. Click "Finish"

12. Load Balancer LB-Demo was created

13. Wait the state "Spawning" to "[[Active]]"

14. Click "LB-Demo"

15. Click Create.

16. Select HTTP

17. Select the Port "80" or Different for you environment

18. Click "Forward".

19. Click Select the "Target group".

PS: We have other tutorial for this, find "How to Create a Target Group"

20. Click "Finish"

21. Click "TG-Demo" to see the instance behind load balance

22. With load balance [[Public IP]] "102.219.126.140", you can see website/app works
