How to access via SSH

Made by Clouds2 with Scribe

This guide provides step-by-step instructions on how to access a server via SSH using Windows PowerShell or a similar tool. It includes information on navigating to the folder containing the private key file, adding permissions if using Linux, and the exact command to enter for accessing the server. By following this guide, users can easily connect to a server securely and efficiently using SSH.



1. Open "Windows PowerShell" or similar

2. Navigate to the folder that contains your private key file.
Hint: If you use Linux, you will need to add permission to this key “chmod 400”.

3. Exemple Type: " ssh -i .\MinhaChavePrivada.pem ubuntu@185.123.123.123 "
" ssh -i .\MyKey.pem ubuntu@MyPublicIP "

4. Accept the fingerprint and you're in.
