How to Create Snapshot

Made by Clouds2 with Scribe

This guide provides step-by-step instructions on how to create a snapshot of an instance. It includes navigating to the appropriate website, selecting the instance, and giving a name to the snapshot. This guide is useful for someone who needs to create a snapshot for backup or cloning purposes.



1. Navigate to https://for-z-mnode.clouds2brasil.com.br/

2. Click "Instances"

3. Select the Instance

4. Click "More"

5. Click "Snapshot"

6. Give a name for this Snapshot

7. Click "Ok"

8. Click "Snapshots"

9. Here you can see all snapshots
